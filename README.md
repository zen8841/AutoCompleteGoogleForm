# AutoCompleteGoogleForm

## HowToUse

1. 下載TamperMonkey到瀏覽器中
2. 打開TamperMonkey的控制臺，點擊**已安裝的使用者腳本**左側的**加號**新增腳本
3. 將[AutoComplete.js](https://github.com/zen8841/AutoCompleteGoogleForm/blob/main/AutoComplete.js)裡的內容取代原本的內容
4. 大功告成
